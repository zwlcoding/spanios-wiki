
# Spanios Wiki

Spanios Wiki is a web application built with Vite and React.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Node.js (v20.x or later)
- pnpm

### Installing

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/spanios-wiki.git
   ```
2. Install NPM packages
   ```sh
   pnpm install
   ```

### Running the application

```sh
pnpm dev
```

This will run the app in development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Available Scripts

In the project directory, you can run:

- `pnpm dev`: Runs the app in the development mode.
- `pnpm build`: Builds the app for production to the `dist` folder.
- `pnpm serve`: Serves the built app in production mode.
- `pnpm test`: Runs the test suite.
- `pnpm format`: Formats the code using Biome.
- `pnpm lint`: Lints the code using Biome.
- `pnpm check`: Checks the code using Biome.
- `pnpm start`: Starts the production server.

## Tech Stack

- [Vite](https://vitejs.dev/) - Next Generation Frontend Tooling
- [React](https://reactjs.org/) - A JavaScript library for building user interfaces
- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework
- [Biome](https://biomejs.dev/) - A toolchain for web projects, designed to replace Babel, ESLint, webpack, Prettier, Jest, and others.
- [pnpm](https://pnpm.io/) - Fast, disk space efficient package manager
